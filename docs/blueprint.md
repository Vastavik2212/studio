# **App Name**: Sweet Gallery

## Core Features:

- Cake Showcase: Display high-quality images of available cakes in a gallery format with filters.
- Detailed Cake View: Provide detailed descriptions, ingredients, and price for each cake.
- Booking Button: Allow users to book a cake by selecting a date and time for cake pickup or delivery.

## Style Guidelines:

- Primary color: Soft coral (#F08080), evoking warmth and sweetness.
- Background color: Light blush (#FAEBD7), a desaturated tint of the primary color, for a subtle and elegant backdrop.
- Accent color: Muted lavender (#D8BFD8) for interactive elements and highlights, providing a delicate contrast.
- Headline font: 'Playfair', a modern serif with an elegant, high-end feel.
- Body font: 'PT Sans', a humanist sans-serif paired with Playfair, used for longer text and descriptions.
- Delicate, hand-drawn style icons representing cake flavors and dietary options.
- Clean and spacious design with a focus on showcasing cake photography.
- Gentle fade-in effects and transitions to enhance user engagement.
